## 개념

**병합 정렬 + 삽입 정렬**을 결합한 안정적인 하이브리드 정렬 알고리즘  
합병 정렬을 기반으로 구현하되, 일정 크기 이하의 부분 리스트에 대해 이진 삽입 정렬을 수행

**목적** : 현실 데이터들의 종류와 상관 없이 최적으로 정렬을 잘 수행하기 위해서 개발 된 방식

---

## 성능 및 특징

- **시간 복잡도**
    - 최선: O(n) - 이미 거의 정렬된 경우
    - 평균: O(n log n)
    - 최악: O(n log n)
- **공간 복잡도**: O(n)
- **안정 정렬**: 동일한 값의 상대적 순서 유지
- **실제 데이터 최적화**: 부분 정렬된 데이터에 특히 효율적
